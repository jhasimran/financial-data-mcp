# Financial Data MCP Server

Plug financial data (FX, crypto, stocks) into AI agents using MCP.

## Features
- Currency conversion (no API key required)
- Crypto price lookup
- Stock data (optional)

## Use Cases
- AI financial assistants
- Budgeting agents
- Trading copilots

## Tech Stack
- Python + FastAPI
- Public APIs (ExchangeRate, CoinGecko)
